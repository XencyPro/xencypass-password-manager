# About XencyPro 🛡️

### Who We Are
**XencyPro** is a small, dedicated development studio focused on building high-performance, secure, and privacy-centric utility tools. We believe that in the modern digital age, every user deserves full ownership of their data.

### The XencyPass Mission
XencyPass was developed to solve the "Cloud Trust" problem. Many password managers store data on their own servers, which are targets for hackers. XencyPass eliminates this risk by:
- Storing data **locally**.
- Using **your** cloud (Google Drive) for backups.
- Using **Premium UI** to make security a beautiful experience.

### Our Values
1. **Privacy First:** If it doesn’t need to be online, it stays offline.
2. **Transparency:** Clear documentation and no hidden trackers.
3. **User Empowerment:** Giving you the tools to manage your own digital vault.
