# Installation & Setup Guide 🚀

### 📦 Prerequisites
- Android 8.0 (Oreo) or higher.
- Biometric hardware (Recommended for the best experience).
- Google Account (Required for cloud backups).

### 📥 Download Instructions
1. Navigate to the **[Releases](https://github.com/XencyPro/xencypass-offline-password-manager/releases)** section.
2. Download the latest `app-release.apk` (or the `.aab` file if you are using a custom installer).

### 🛠️ Setup Steps
1. **Initial Launch:** Set up your **Master PIN**. Choose something memorable but secure.
2. **Biometrics:** Enable Fingerprint or Face ID for faster and more secure access.
3. **Cloud Sync:** Go to Settings -> Backup and sign in with your Google account. This ensures you can restore your data if you switch phones.
4. **Importing Data:** Use the "Add Item" button to start saving your 8-digit codes, recovery phrases, and passwords.

### ⚠️ Security Warning
**NEVER** share your `xencypass_backup.json` (or backup file) with anyone. Even though it is encrypted, it is the most sensitive file of your application.
